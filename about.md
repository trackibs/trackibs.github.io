TrackIBS is an open source project to help people track their IBS Symptoms

Please feel free to contribute by :

* Creating a feature request - [TrackIBS Issues](https://github.com/trackibs/TrackIBS/issues)

* Contributing a Pull Request - [TrackIBS Pull Request](https://github.com/trackibs/TrackIBS/pulls)

* Join the community on - [TrackIBS Reddit](https://www.reddit.com/r/trackibs)

* Join the Chat Room - [TrackIBS Chat Room](https://s.reddit.com/channel/29674584_9a9f8d598e5a5fdcb2a0846e6421f823fc24e00c)
